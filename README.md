This is a timeline based on the [Timeglider] (https://timeglider.com/widget/index.php) adjusted for my own needs. I built that for the course of "History of Electroacoustic Music" during my studies in Sibelius Academy, in order to put events in a timeline manually by the user.



An example can be found here:

http://gotsopoulos.com/timeline/


You can add new events through this form:
http://gotsopoulos.com/timeline/form2.php


All the events are added to an XML file by running the following PHP script
http://gotsopoulos.com/timeline/concatenate_events.php


Each event has a unique ID that is shown in the even description

An event can be deleted using its unique ID through the following form:

http://gotsopoulos.com/timeline/delete_form.html

Then the XML list has to be reconstructed using the same script as earlier:

http://gotsopoulos.com/timeline/concatenate_events.php
